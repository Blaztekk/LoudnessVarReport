Outil d'analyse des différences sonores entre un pool de fichiers mp3/mp4/mkv
**PS1 n'est plus maintenu, utiliser python**

## Prérequis
- Ps / Py
- Ffmpeg
## Usage

1. Démarrer `py .\New-SoundReport.py`
2. sélectionner le dossier ou se trouve vos médias
3. Etudier le rapport.htlm. Un .csv est dispo pour partager plus facilement les résultats.
